#Projet NodeJs NoSQL

Projet troisiéme année à Ingésup


##Install

Installez nodeJS, Redis et mongoDB

Ouvrez 3 terminals
  - Dans le premier tapez
      Mongod.exe

  - Dans le second tapez
      redis-cli

  - Dans le troisième taper
      cd 'votre chemin' + projet_node_nosql
      npm install
      npm start
